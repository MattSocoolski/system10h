## ANTY-AI CONTENT - ZASADY NATURALNEGO PISANIA

### FUNDAMENTALNA ZASADA:
**Tekst ma brzmieć jakby pisał go człowiek, nie AI. Żadnych oznak sztuczności.**

### ZAKAZANE SŁOWA I FRAZY:

**Puste wyrazy-wytrychy:**
- kluczowy, istotny, fundamentalny, wszechstronny
- niezwykły, wyjątkowy, fascynujący, niezaprzeczalny
- stanowi świadectwo, odgrywa kluczową rolę
- nie sposób przecenić, trudno nie docenić

**Konstrukcje AI:**
- "Nie tylko X, ale także Y" -> Napisz normalnie co chcesz powiedzieć
- "...podkreślając tym samym...", "...ukazując jednocześnie..."
- "Co więcej", "Co ciekawe", "Co istotne" na początku zdań
- "Warto zauważyć, że...", "Należy podkreślić, że..."
- "W dzisiejszych czasach...", "We współczesnym świecie..."

**Struktury promocyjne:**
- "Zanurz się w świat...", "Odkryj moc...", "Wyrusz w podróż..."
- "Przekonaj się sam...", "Dołącz do grona..."

### ZAKAZANE FORMATOWANIE:

- Wielkie Litery W Każdym Słowie Tytułu
- Nadużywanie **pogrubień** i *kursywy*
- Emoji w formalnych tekstach (w social OK z umiarem)
- Nadmiar pauzy - w jednym akapicie
- Strzałki typu -> (używaj myślników lub dwukropków)
- Automatyczne "Podsumowanie" na końcu
- Sekcje "Kluczowe wnioski" bez potrzeby

### JAK PISAĆ NATURALNIE:

1. **Krótkie zdania** - max 15-20 słów
2. **Aktywny głos** - "Zrobiłem" nie "Zostało zrobione"
3. **Konkretne liczby** - "83 klientów" nie "wielu klientów"
4. **Proste słowa** - "użyj" nie "wykorzystaj", "zrób" nie "przeprowadź"
5. **Jedno zdanie = jedna myśl**
6. **Bez wodolejstwa** - każde słowo musi mieć cel

### QUICK CHECKLIST PRZED PUBLIKACJĄ:

```
[] Czy usunąłem "kluczowy/istotny/fundamentalny"?
[] Czy nie ma "...podkreślając tym samym..."?
[] Czy nie ma "Nie tylko X, ale także Y"?
[] Czy formatowanie jest oszczędne (mało bold, zero emoji w formalnym)?
[] Czy nie dodałem "Podsumowania" którego nikt nie prosił?
[] Czy tekst brzmi jak pisany przez człowieka?
```

**ZASADA KOŃCOWA:** Przeczytaj tekst na głos. Jeśli brzmi sztucznie - przepisz.
