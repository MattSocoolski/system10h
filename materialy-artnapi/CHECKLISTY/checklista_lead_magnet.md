# SUPER SZCZEGÓŁOWA CHECKLISTA: Strona Lead Magnet + Popup + Thank You Page

## 📄 STRONA ZAPISU NA LEAD MAGNET (LANDING PAGE)

### NAGŁÓWEK I SEKCJA HERO

Nagłówek (Headline)

- [ ] Maksymalnie 10 słów

- [ ] Zawiera konkretną korzyść lub rezultat

- [ ] Używa liczb gdy możliwe ("5 kroków", "30 dni", "350 zł")

- [ ] Odnosi się bezpośrednio do problemu grupy docelowej

- [ ] Formuła: "[Oferta] dla [kto] żeby [rezultat] bez [problem]"

- [ ] Font: duży (min. 32px desktop, 24px mobile), pogrubiony

- [ ] Kolor: kontrastowy, dobrze czytelny

Podtytuł (Subheadline)

- [ ] 1-2 zdania maksymalnie

- [ ] Rozszerza obietnicę nagłówka

- [ ] Precyzuje dla kogo jest oferta

- [ ] Podkreśla unikalność/natychmiastowość wartości

- [ ] Font: mniejszy niż nagłówek (18-22px)

Hero Image/Video

- [ ] Mockup lead magnetu (okładka PDF, zrzut ekranu szablonu)

- [ ] Wysokiej jakości (min. 1200px szerokości)

- [ ] Format WebP lub optymalizowany JPG (<200KB)

- [ ] Alt text opisowy dla SEO

- [ ] Pokazuje produkt w kontekście użycia

- [ ] NIE generyczne zdjęcia stockowe

- [ ] Opcjonalnie: krótkie wideo (max 60 sekund) z auto-play bez dźwięku

### OPIS I KORZYŚCI

Krótki opis oferty

- [ ] 1-2 zdania czym dokładnie jest lead magnet

- [ ] Jasne określenie formatu (checklista/szablon/e-book/guide)

- [ ] Podkreślenie praktycznego zastosowania

- [ ] Prosty język bez żargonu

Lista konkretnych korzyści (Bullet Points)

- [ ] 3-5 punktów (nie więcej!)

- [ ] Każdy punkt = konkretny rezultat/korzyść

- [ ] Struktura: "Czasownik + konkretna korzyść"

  - ✅ "Zaoszczędzisz 5 godzin tygodniowo dzięki gotowemu szablonowi"

  - ✅ "Unikniesz 7 najczęstszych błędów które kosztują klientów"

  - ✅ "Zwiększysz konwersję o 25% stosując sprawdzone techniki"

- [ ] Używaj ikon/checkmarków przed każdym punktem

- [ ] Font czytelny (min. 16px)

- [ ] Odpowiednia wysokość linii (line-height: 1.6)

### FORMULARZ OPT-IN

Pola formularza

- [ ] Tylko adres e-mail (ZALECANE)

- [ ] Ewentualnie: Imię + E-mail (maksimum!)

- [ ] NIE DODAWAJ: nazwisko, telefon, firma (chyba że absolutnie konieczne dla B2B)

- [ ] Każde pole ma:

  - [ ] Wyraźną etykietę (label)

  - [ ] Placeholder text jako przykład

  - [ ] Walidację w czasie rzeczywistym

  - [ ] Komunikaty błędów po polsku

  - [ ] Autofocus na pierwszym polu

  - [ ] Autofill enabled (autocomplete="email")

Wizualny design formularza

- [ ] Wyraźnie wydzielony box (obramowanie lub cień)

- [ ] Kontrastowe tło (np. lekki kolor vs białe tło strony)

- [ ] Pola wystarczająco duże (min. 44px wysokości dla mobile)

- [ ] Odstępy między polami (min. 16px)

- [ ] Responsive na mobile (100% szerokości na małych ekranach)

Zdanie wprowadzające nad formularzem

- [ ] "Wpisz swój e-mail, aby otrzymać [nazwa lead magnetu]:"

- [ ] Lub: "Otrzymaj natychmiastowy dostęp - podaj e-mail:"

- [ ] Font: 16-18px, medium weight

Zgody RODO

- [ ] Checkbox wymagany: "Zgadzam się na przetwarzanie danych osobowych"

- [ ] Link do polityki prywatności (otwiera się w nowym oknie)

- [ ] Opcjonalny checkbox: zgoda marketingowa (jeśli wymaga prawo)

- [ ] Font małych literek: 12-14px, ale czytelny

Zapewnienie o prywatności

- [ ] Tekst pod formularzem: "Szanujemy Twoją prywatność. Bez spamu, gwarantujemy."

- [ ] Lub: "Nie sprzedajemy danych. Możesz wypisać się w każdej chwili."

- [ ] Opcjonalnie ikona kłódki 🔒

- [ ] Kolor: szary, mniejszy font (14px)

### PRZYCISK CTA

Tekst na przycisku

- [ ] Zorientowany na akcję + korzyść

- [ ] Pierwsza osoba: "Pobierz MOJĄ checklistę"

- [ ] Konkretny czasownik:

  - "Pobierz darmowy szablon"

  - "Wyślij mi przewodnik"

  - "Chcę dostać dostęp"

  - "Tak, chcę pobrać!"

- [ ] NIE używaj: "Wyślij", "Prześlij", "Submit", "Kliknij tutaj"

- [ ] Długość: 2-5 słów

Wizualny design przycisku

- [ ] Kolor: maksymalnie kontrastowy z tłem strony

- [ ] Rozmiar: duży, niemożliwy do przeoczenia

  - Desktop: min. 180px szerokości, 50px wysokości

  - Mobile: min. 280px szerokości, 56px wysokości

- [ ] Font: pogrubiony (bold), 16-18px

- [ ] Zaokrąglone rogi (border-radius: 4-8px)

- [ ] Stan hover: zmiana koloru/cienia

- [ ] Stan kliknięcia: animacja/feedback

- [ ] Opcjonalnie ikona: ⭳ (pobierz) lub ➡️ (dalej)

- [ ] Pusta przestrzeń wokół (min. 30px margin)

Umiejscowienie CTA

- [ ] Główny przycisk: "above the fold" (widoczny bez scrollowania)

- [ ] Powtórzenie: na dole strony (jeśli dłuższa strona)

- [ ] Na mobile: sticky button na dole ekranu (opcjonalnie)

Tekst dodatkowy pod przyciskiem

- [ ] "Natychmiastowy dostęp. Pobierz w 30 sekund."

- [ ] Lub: "Darmowe. Bez karty kredytowej."

- [ ] Lub: "Otrzymasz link na e-mail w ciągu 2 minut"

- [ ] Font: 14px, italic, kolor szary

### SOCIAL PROOF (DOWODY SPOŁECZNE)

Opinie/Testimoniale

- [ ] 1-3 konkretne referencje

- [ ] Każda opinia zawiera:

  - [ ] Zdjęcie osoby (64x64px minimum, okrągłe)

  - [ ] Imię i nazwisko

  - [ ] Stanowisko/firma/lokalizacja

  - [ ] Konkretna korzyść (nie ogólniki!)

  - [ ] 2-3 zdania maksymalnie

- [ ] Cytat w cudzysłowie lub wyróżniony box

- [ ] Opcjonalnie: 5 gwiazdek ⭐⭐⭐⭐⭐

Liczby i statystyki

- [ ] "Dołącz do 5 247 osób, które już pobrały"

- [ ] Lub: "Ponad 10 000 pobrań"

- [ ] Lub: "98% użytkowników ocenia na 5/5"

- [ ] Duży, wyróżniony font dla liczby

- [ ] Umieszczenie: pod formularzem lub przy nagłówku

Logotypy firm/mediów

- [ ] Sekcja "Zaufali nam:" lub "Pisali o nas:"

- [ ] 3-6 logotypów w jednej linii

- [ ] Grayscale lub niski kontrast (żeby nie rozpraszać)

- [ ] Równej wysokości (30-40px)

- [ ] Responsive: 2 kolumny na mobile

Element autorytetu

- [ ] Zdjęcie autora (okrągłe, 80-100px)

- [ ] Imię, nazwisko, tytuł/credentials

- [ ] 1 zdanie o doświadczeniu

- [ ] Umieszczenie: sidebar lub sekcja osobna

## 🪟 POPUP (EXIT-INTENT)

### KIEDY SIĘ POKAZUJE

Trigger (wyzwalacz)

- [ ] Exit-intent: kursor kieruje się do zamknięcia karty

- [ ] Lub: po 45-60 sekundach na stronie

- [ ] Lub: scroll 50%+ strony w dół

- [ ] NIE: natychmiastowo po wejściu

- [ ] Pokazuje się tylko raz na sesję

- [ ] Cookie żeby nie denerwować (7-30 dni)

### ZAWARTOŚĆ POPUPU

Nagłówek

- [ ] Krótki, maksymalnie 6-8 słów

- [ ] Inny niż na głównej stronie

- [ ] Opcje:

  - "Zaczekaj! Nie odchodź z pustymi rękami"

  - "Ostatnia szansa na darmowy [lead magnet]"

  - "Zanim wyjdziesz - weź prezent"

- [ ] Font: duży (24-28px), pogrubiony

Tekst opisowy

- [ ] 1-2 zdania maksymalnie

- [ ] Podkreślenie wartości/korzyści

- [ ] Prosty język

Mini-formularz

- [ ] TYLKO pole e-mail (bez imienia)

- [ ] Placeholder: "Twój najlepszy e-mail"

- [ ] Duże pole (min. 280px szerokości)

Przycisk CTA

- [ ] Tekst akcji: "Tak, wyślij mi to!" lub "Chcę pobrać!"

- [ ] Kontrastowy kolor

- [ ] Pełna szerokość pola e-mail

- [ ] Wysokość min. 48px

Przycisk zamykający

- [ ] Wyraźny X w prawym górnym rogu

- [ ] Min. 32x32px (touch-friendly)

- [ ] Kolor: ciemny szary

- [ ] Opcjonalnie: tekst pod formularzem "Nie, dziękuję" jako link

### DESIGN POPUPU

Rozmiar i układ

- [ ] Szerokość: 480-600px desktop

- [ ] Wysokość: auto (nie za wysoki!)

- [ ] Mobile: 90% szerokości ekranu

- [ ] Wycentrowany na ekranie

- [ ] Tło: białe lub jasny kolor

- [ ] Padding: 40px wszystkie strony

Overlay (tło)

- [ ] Ciemne tło (rgba(0,0,0,0.7))

- [ ] Blur lub zaciemnienie reszty strony

- [ ] Kliknięcie w overlay = zamknięcie popupu

Obrazek/ikona (opcjonalnie)

- [ ] Mała ikona prezentu 🎁 lub mockup lead magnetu

- [ ] Max 150x150px

- [ ] Umieszczenie: góra lub lewa strona

Elementy social proof w popup

- [ ] Mikro-testimonial: "Najlepszy materiał jaki dostałam!" - Anna K.

- [ ] Lub liczba: "Już 3 284 osoby pobrały"

- [ ] Lub gwiazdki: ⭐⭐⭐⭐⭐ 4.9/5

- [ ] Font: mały (12-14px), italic

### WARIANTY POPUPU

Wariant A: Dwuetapowy popup

- [ ] Krok 1: Tylko przycisk "Tak, chcę darmowy [lead magnet]!"

- [ ] Krok 2: Dopiero formularz z e-mailem

- [ ] Animacja przejścia między krokami

Wariant B: Rabat/bonus popup

- [ ] "Zanim wyjdziesz: 20% zniżki na [produkt]"

- [ ] Kod rabatowy automatycznie aplikowany

- [ ] Dla e-commerce B2C

Wariant C: Kontekstowy popup

- [ ] Różna treść w zależności od podstrony

- [ ] Artykuł o SEO → popup z checklistą SEO

- [ ] Artykuł o social media → popup z szablonem postów
