## D. BLIND SPOTS

### Dlaczego ludzie myślą że "sami ogarną"
- **Ego techniczne:** "Umiem kodować = umiem wszystko tech." Ale Bliźniak to metodologia, nie tech.
- **Koszty zatopione:** "Płacimy za ChatGPT = mamy AI." Narzędzie ≠ system.
- **Brak widoczności:** Właściciel nie widzi ile czasu handlowcy tracą, bo sam nie sprzedaje.

### Dunning-Kruger w AI
"Zainstalowałem ChatGPT = mam AI" to jak "Mam Excela = mam ERP". Software house powinien to rozumieć lepiej niż ktokolwiek.

### Prompt vs System
Prompt = jednorazowe pytanie. System = 240 pól DNA + 16 scenariuszy + pamięć + routing + Protokół Wiarygodności + stylometria.

### Ola jest gatekeeper, nie właściciel
Sprzedawaj WARTOŚĆ Oli (oszczędność czasu). Daj Oli NARZĘDZIA do przekonania właściciela (PDF, ROI). Nie walcz z właścicielem bezpośrednio.
