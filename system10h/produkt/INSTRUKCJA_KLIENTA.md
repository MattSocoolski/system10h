# BLIŹNIAK BIZNESOWY — Quick Start

Masz swój plik `dna.md`. Czas uruchomić Bliźniaka. Wybierz swoją platformę.

---

## Opcja A: Claude.ai (przeglądarka)

### Krok 1: Otwórz Claude
Wejdź na **claude.ai** → Zaloguj się → Kliknij **Projects** (lewy panel) → **New Project**

Nazwij projekt: **Bliźniak Biznesowy**

### Krok 2: Wklej pliki
W sekcji **Project Knowledge** dodaj dwa pliki:

1. **`dna.md`** — Twoje Biznesowe DNA (dostałeś/aś ode mnie)
2. **`SKILL.md`** — Instrukcja dla Bliźniaka (dostałeś/aś ode mnie)

Kliknij "Add content" → wklej zawartość każdego pliku.

### Krok 3: Napisz "cześć"
Bliźniak pokaże Ci menu z 16 opcjami. Wybierz numer lub napisz po swojemu.

---

## Opcja B: Claude Code (terminal)

### Krok 1: Utwórz folder projektu
```bash
mkdir blizniiak-biznesowy
cd blizniiak-biznesowy
```

### Krok 2: Wrzuć 3 pliki
Skopiuj do folderu pliki które dostałeś/aś ode mnie:
- **`CLAUDE.md`** — plik systemowy (instrukcje dla AI)
- **`dna.md`** — Twoje Biznesowe DNA
- **`SKILL.md`** — instrukcja Bliźniaka (16 workflow'ów)

### Krok 3: Uruchom Claude Code
```bash
claude
```
Bliźniak załaduje Twoje DNA i pokaże menu. Gotowe.

**Bonus:** W Claude Code Bliźniak automatycznie robi backup plików przed każdą zmianą.

## Deep Research (opcja 13) — jak działa?

1. Wybierz **13** lub napisz "deep research"
2. Bliźniak zaproponuje 5 tematów na podstawie Twojego DNA
3. Wybierz temat (lub podaj swój)
4. Bliźniak wygeneruje **gotowy prompt** do skopiowania
5. Wklej go do **Gemini Deep Research** lub **Claude.ai**
6. Wróć z wynikami — Bliźniak pomoże wyciągnąć wnioski

---

## Higiena folderu (opcja 14) — porządek w plikach

Bliźniak sprawdza:
- Czy Twoje pliki (dna.md, CLAUDE.md, SKILL.md) nie są za długie
- Czy każda sekcja dna.md mieści się w limicie
- Czy w folderze nie ma śmieci/duplikatów
- Czy backup/ nie jest przepełniony (max 10 plików)

Napisz "higiena" lub "14" — dostaniesz raport z rekomendacjami.

## Rozwój biznesu (opcja 15) — strategia wzrostu

6 trybów do wyboru:

| Tryb | Co robi |
|------|---------|
| A. Audyt kwartału | Co zadziałało, co nie — na bazie Twoich danych |
| B. Nowe segmenty | Kto jeszcze mógłby kupić Twój produkt? |
| C. Mapa upsell | Co sprzedać obecnym klientom? |
| D. Optymalizacja cen | Czy Twoja cena jest optymalna? |
| E. System poleceń | Zbuduj maszynę do referrali |
| F. Plan 90 dni | 3 priorytety na następny kwartał |

Napisz "rozwój" lub "15" — wybierzesz tryb.

---

## Radar szans (opcja 16) — ukryte przychody

Bliźniak analizuje CAŁE Twoje DNA + stan operacyjny i szuka:
- **Top szansa** — największa okazja przychodowa z estymacją PLN/h
- **1 rzecz do porzucenia** — co zabiera czas bez zwrotu
- **1 wild card** — nieoczywisty pomysł z potencjałem

Napisz "radar" lub "16" — dostaniesz analizę.
