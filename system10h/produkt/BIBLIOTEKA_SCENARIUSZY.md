# BIBLIOTEKA SCENARIUSZY BLIŹNIAKA BIZNESOWEGO

16 gotowych scenariuszy. Każdy referencuje `dna.md` jako źródło danych.

---

## WF1: Odpowiedź na maila (Email Reply)

**KIEDY:** Dostajesz maila od klienta i chcesz szybko odpisać.
**JAK:** Wklej treść maila i napisz "odpisz" lub "odpowiedz na tego maila".
**CO SPRAWDZA W DNA:** S5 (Twój styl), S6 (battle cards jeśli obiekcja), S4 (ceny jeśli pytanie o cenę)
**PRZYKŁAD:**
```
Dostałem takiego maila:

"Cześć Mateusz, dzięki za spotkanie. Oferta wygląda ok ale chcielibyśmy
jeszcze porównać z innym dostawcą. Daj mi tydzień."

Odpisz.
```

## WF2: Tworzenie oferty (Offer Creation)

**KIEDY:** Klient prosi o ofertę lub wycenę.
**JAK:** Napisz "przygotuj ofertę dla [klient/firma]" + podaj co wiesz o kliencie.
**CO SPRAWDZA W DNA:** S3 (produkty, ceny, USP), S4 (warunki), S5 (styl)
**PRZYKŁAD:**
```
Przygotuj ofertę dla firmy XYZ.
Branża: e-commerce, 15 osób.
Potrzebują: automatyzacji follow-upów i cold outreach.
Budżet: nie podali.
```

## WF3: Obrona ceny — Bad Cop (Price Defense)

**KIEDY:** Klient mówi "za drogo", "chcę rabat", porównuje z konkurencją.
**JAK:** Opisz sytuację lub wklej wiadomość klienta.
**CO SPRAWDZA W DNA:** S6 (battle cards), S3 (USP, dowody), S4 (polityka rabatowa)
**PRZYKŁAD:**
```
Klient napisał: "Marek, lubię Wasz produkt ale 2500 to za dużo.
Konkurencja ma podobne za 1500. Możemy dogadać się na 1800?"

Jak odpowiedzieć? Nie chcę schodzić z ceny.
```

## WF4: Follow-up

**KIEDY:** Klient zamilkł — nie odpisuje na maila/ofertę.
**JAK:** Napisz "follow-up" + ile minęło czasu + co było ostatnim kontaktem.
**CO SPRAWDZA W DNA:** S5 (styl), S7 (procedury follow-up)
**PRZYKŁAD:**
```
Follow-up do Anny z firmy ABC.
Wysłałem ofertę 5 dni temu, zero odpowiedzi.
Przed ofertą mieliśmy dobry Zoom.
```

## WF5: Windykacja (Debt Collection)

**KIEDY:** Klient nie zapłacił faktury po terminie.
**JAK:** Napisz "faktura po terminie" + podaj ile dni i kwotę.
**CO SPRAWDZA W DNA:** S7 (windykacja 4-etapowa), S5 (styl)
**PRZYKŁAD:**
```
Klient: Firma DEF
Faktura: FV/2026/02/001 na 8 500 PLN
Po terminie: 8 dni
Kontakt wcześniej: wysłałem przypomnienie 3 dni temu, bez odpowiedzi.
```

## WF6: Outreach — Cold/Warm (Zaczepki)

**KIEDY:** Chcesz napisać pierwszą wiadomość do nowego prospekta.
**JAK:** Napisz "cold outreach" lub "napisz zaczepkę" + podaj co wiesz o osobie/firmie.
**CO SPRAWDZA W DNA:** S2 (klient, bóle), S3 (USP), S5 (styl)
**PRZYKŁAD:**
```
Cold outreach na LinkedInie.
Osoba: Jan Kowalski, CEO w firmie GHI (software house, 20 osób).
Widziałem jego post o problemach ze skalowaniem sprzedaży.
Chcę zaproponować rozmowę o automatyzacji.
```

## WF7: Research prospekta (Prospect Research)

**KIEDY:** Przed pierwszym kontaktem lub spotkaniem — chcesz wiedzieć więcej o firmie/osobie.
**JAK:** Napisz "prześwietl firmę" lub "research" + podaj nazwę i co masz.
**CO SPRAWDZA W DNA:** S2 (avatar, anty-avatar), S3 (konkurencja)
**PRZYKŁAD:**
```
Prześwietl firmę JKL.
Strona: www.jkl.pl
Branża: chyba IT consulting.
Mam jutro z nimi call.
```

---

## WF8: Przygotowanie na spotkanie (Meeting Prep)

**KIEDY:** Masz nadchodzące spotkanie z klientem/prospektem.
**JAK:** Napisz "przygotuj mnie na spotkanie" + podaj z kim i o czym.
**CO SPRAWDZA W DNA:** S6 (obiekcje), S4 (ceny), S2 (bóle klienta), S3 (oferta)
**PRZYKŁAD:**
```
Jutro Zoom z Anną z firmy MNO.
Temat: prezentacja naszego produktu.
Firma: e-commerce, 30 osób, mają problem z konwersją leadów.
Cel: zamknąć na pilotaż.
```

## WF9: Content (LinkedIn, Case Study)

**KIEDY:** Chcesz napisać post na LinkedIn, case study, artykuł.
**JAK:** Napisz "post LinkedIn o [temat]" lub "case study [klient]".
**CO SPRAWDZA W DNA:** S1 (profil, origin story), S3 (dowody), S5 (styl)
**PRZYKŁAD:**
```
Napisz post na LinkedIn.
Temat: Jak AI zaoszczędziło mi 10h/tydzień w sprzedaży B2B.
Ton: praktyczny, bez buzzwordów, z konkretnymi liczbami.
```

## WF10: Briefing pipeline'u (Pipeline Analysis)

**KIEDY:** Rano, gdy chcesz wiedzieć co robić. Lub tygodniowy przegląd.
**JAK:** Napisz "co robić dziś?" lub "briefing" + podaj aktywne leady.
**CO SPRAWDZA W DNA:** S8 (workflow, KPI), S2 (klient)
**PRZYKŁAD:**
```
Briefing na dziś. Moje aktywne leady:
1. Anna (MNO) — wysłałem ofertę 3 dni temu, cisza
2. Jan (GHI) — Zoom jutro
3. Piotr (PQR) — zapłacił, czeka na onboarding
4. Ewa (STU) — cold outreach wysłany tydzień temu, zero
```

## WF11: Propozycja handlowa (Proposal)

**KIEDY:** Klient potrzebuje formalnej propozycji/dokumentu do podjęcia decyzji.
**JAK:** Napisz "przygotuj propozycję" + podaj kontekst.
**CO SPRAWDZA W DNA:** S3 (oferta, ceny, dowody), S4 (warunki), S5 (styl)
**PRZYKŁAD:**
```
Przygotuj propozycję handlową dla firmy VWX.
Branża: produkcja, 50 osób.
Potrzebują: pełnego systemu AI dla działu handlowego (4 osoby).
Budżet: do 10k PLN.
Deadline decyzji: koniec miesiąca.
```

## WF12: Obsługa reklamacji (Complaint Handling)

**KIEDY:** Klient jest niezadowolony, złożył reklamację, napisał wściekłego maila.
**JAK:** Wklej wiadomość klienta lub opisz sytuację. Napisz "reklamacja" lub "klient wściekły".
**CO SPRAWDZA W DNA:** S7 (procedury reklamacji), S5 (styl)
**PRZYKŁAD:**
```
Klient napisał:

"Mateusz, ten system nie działa jak obiecywałeś. Asystent pisze
generyczne odpowiedzi, zero personalizacji. Chcę zwrot pieniędzy."

Co odpowiedzieć?
```

## WF15: Rozwój biznesu (Business Growth)

**KIEDY:** Chcesz zaplanować wzrost na następny kwartał.
**JAK:** Napisz "rozwój" lub "plan na Q2" lub "jak zwiększyć przychód?".
**CO SPRAWDZA W DNA:** S1 (profil), S2 (klient), S3 (oferta), S4 (zasady), S7 (procedury), S8 (workflow)
**PRZYKŁAD:**
```
Plan na Q2. Chcę zwiększyć przychód o 30%.
Obecni klienci: 12, średnia wartość: 3000 PLN.
Nowe segmenty? Upsell? Ceny?
```

## WF16: Radar szans (Opportunity Radar)

**KIEDY:** Szukasz ukrytych szans przychodowych — co pomijasz, gdzie jest kasa.
**JAK:** Napisz "radar" lub "co pomijam?" lub "gdzie jest kasa?".
**CO SPRAWDZA W DNA:** S1-S8 + stan.md
**PRZYKŁAD:**
```
Radar. Co pomijam? Gdzie jest kasa której nie widzę?
Mam 8 klientów, robię głównie usługi wdrożeniowe.
```
