# BLIŹNIAK BIZNESOWY — Szybki Start

Masz plik `dna.md`. Czas uruchomić Bliźniaka. 3 kroki, 5 minut.

---

## Co to jest Bliźniak Biznesowy?

To asystent AI, który zna Twój biznes, Twoje ceny, Twój styl komunikacji. Odpowiada klientom Twoim głosem, przygotowuje wyceny wg Twojego cennika i broni Twojej marży. Wszystko na podstawie Twojego Biznesowego DNA.

---

## Krok 1: Otwórz Claude

Wejdź na **claude.ai** → Zaloguj się → Kliknij **Projects** (lewy panel) → **New Project**

Nazwij projekt: **Bliźniak Biznesowy**

---

## Krok 2: Wgraj pliki

W sekcji **Project Knowledge** dodaj trzy pliki:

1. **`CLAUDE.md`** — Instrukcje dla systemu (dostałeś ode mnie)
2. **`dna.md`** — Twoje Biznesowe DNA (dostałeś ode mnie)
3. **`SKILL.md`** — 15 scenariuszy pracy (dostałeś ode mnie)

Kliknij "Add content" → wgraj każdy plik.

---

## Krok 3: Zacznij pisać

Otwórz czat w projekcie i napisz cokolwiek:

- `"odpisz"` + wklej maila od klienta
- `"wycena wizytówek 1000 szt. dla firmy budowlanej"`
- `"co robić dziś?"`
- `"klient mówi za drogo"`

Bliźniak zna Twój biznes. Pisze Twoim głosem. Broni Twojej marży.

---

## Główne komendy

| Co potrzebujesz | Co napisać |
|-----------------|------------|
| Odpisać klientowi | `odpisz` + wklej maila |
| Przygotować wycenę | `wycena` / `oferta` |
| Obronić cenę | `za drogo` / `klient chce rabat` |
| Przypomnieć klientowi | `follow-up` + ile minęło |
| Ściągnąć należność | `windykacja` / `faktura` |
| Napisać zaczepkę | `zaczepka` / `outreach` |
| Prześwietlić firmę | `prześwietl` + nazwa firmy |
| Przygotować się na spotkanie | `spotkanie` + z kim |
| Stworzyć ofertę mailową | `content` / `oferta mailowa` |
| Przegląd zleceń | `co robić` |
| Zaproponować pakiet | `propozycja` / `pakiet` |
| Rozwiązać reklamację | `reklamacja` |

Albo po prostu pisz swoimi słowami — AI sam rozpozna, czego potrzebujesz.

## Wskazówki

- **Twoje DNA = Twoje zasady.** AI stosuje Twoje ceny i styl. Jeśli coś nie tak — zaktualizuj `dna.md`.
- **Im więcej szczegółów** podasz w zapytaniu, tym lepsza odpowiedź.
- **Menu** — napisz `menu` żeby zobaczyć wszystkie 15 opcji.
